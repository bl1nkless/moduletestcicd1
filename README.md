# Module Test CI/CD

Початкова ініціалізація проєкту для поетапної розробки Python-скрипта з тестами та CI.

## План

1. Ініціалізація структури проєкту
2. Реалізація основної функції фільтрації рядків
3. Додавання unit-тестів на `pytest`
4. Генерація HTML-звіту через `pytest-html`
5. Налаштування GitHub Actions
